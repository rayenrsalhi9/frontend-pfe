Menu Stucture

We creating navigations from data dynamically, configuration can be found at src/app/configs/nav.config.ts here is our Navigation metadata looks like:


    

        
            

                path
                The route path navigate to.
            

            

                title
                Title of the navigation item.
            

            

                translateKey
                Translate key the navigation item. 
            

            

                type
                Navigation item type. 
            

            

                iconType
                Icon type of the navigation item. 
            

            

                icon
                Icon name of the navigation item according to iconType. 
            

            

                submenu
                Array of navigation items to create child items, value will be [] if there is no sub menu.
            

            

                key
                Unique id of the navigation item.
