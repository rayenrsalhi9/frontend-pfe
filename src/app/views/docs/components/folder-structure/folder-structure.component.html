app/configs

configs folder basically contains app configuration related files, such as app, navigation, chart & i18n.

app/i18n

configs folder included all language retaled files such as translate key & language entry files.

app/layout

This directory contains all the template layouts & related component.

app/mock-data

This is the directory for display purpose mock data.

app/routes

This directory contains the main route tree for each layout.

app/shared

Shared directory included anything re-usable such as components, services and pipes. There is also SharedModule.ts which imports & exports modules that commonly use on develope.

app/store

We use ngxs to maintain our states, this directory contains all states related files.

app/views

This directory included all exmaple apps and ready to use pages that display oon demo.
